1. Скачайте файл `autojudge.py`
2. При первом запуске программа потребует ввести ваш API токен. Вот инструкция, как его получить:

   2. Зайдите в раздел API Keys

## Использование
Для того, чтобы отправить файл в ejudge, просто выполните `python autojudge.py <file>`. При первой отправке файла из директории программа спросит вас о номере контеста, в который отправлять файл. В дальнейшем все файлы из этой директории будут автоматически отправляться в этот контест.
Программа попытается определить номер задачи по названию файла. Например, файлы `A.py` или `C.js` программа отправит в задания A и C соответственно. Если таких задач в контесте нет, или программа не смогла определить номер задачи, то вам будет предложено выбрать задачу из списка доступных.
После отправки программа выведет номер посылки и вердикт тестирующей системы.
## Файлы
Помимо исполняемого файла программа также использует директорию `~/.config/autojudge/`, в которой находятся конфигурационные файлы.
В файле `config.json` хранится ваш API токен в незашифрованном виде и таблица соответствий между директориями и номерами соответствующих контестов.
